## Section G: Profit Factor Calculation

### G.1 Core PF Function
```python
# chaos_gpu_training.py:560-615
def calculate_profit_factor(predictions, returns):
    """
    PF = Gross Profit / Gross Loss

    WITH TRADE FREQUENCY PENALTY:
    - If trade_ratio < MIN_TRADE_RATIO (10%), penalize PF
    """
    positions = convert_to_positions(predictions)  # 0,1,2 -> -1,0,+1

    n_trades = np.sum(positions != 0)
    trade_ratio = n_trades / len(positions)

    if trade_ratio < MIN_TRADE_RATIO:
        penalty = trade_ratio / MIN_TRADE_RATIO
        return 0.5 * penalty  # Returns 0-0.5 for low trade models

    pnl = positions * returns
    gross_profit = np.sum(pnl[pnl > 0])
    gross_loss = np.abs(np.sum(pnl[pnl < 0]))

    pf = gross_profit / gross_loss
    return pf
```

### G.2 Position Conversion
```python
# Predictions: 0=short, 1=flat, 2=long
# Positions:  -1=short,  0=flat, +1=long
positions = predictions.astype(np.float64) - 1.0
```

### M.3 Batched Inference (OOM Prevention)
```python
def batched_inference(model, X_tensor, batch_size=2048):
    """Process validation in batches to prevent OOM on Conv1D models."""
    model.eval()
    outputs = []
    with torch.no_grad():
        for i in range(0, len(X_tensor), batch_size):
            batch = X_tensor[i:i+batch_size]
            out = model(batch.to(device))
            outputs.append(out.cpu())  # Move to CPU immediately
    return torch.cat(outputs, dim=0)
```
